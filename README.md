**Install Dependencies**

```
npm install
```

**Build Project**

```
npm run build
```

**Start Server**

```
npm start
```

**Run Tests**

```
npm test
```

**Start in Developer Mode**

```
npm run start:watch
```

**Create a Component**

```
npm run components:create -- path/NameOfComponent
npm run components:create -- --styled path/NameOfComponent
```
